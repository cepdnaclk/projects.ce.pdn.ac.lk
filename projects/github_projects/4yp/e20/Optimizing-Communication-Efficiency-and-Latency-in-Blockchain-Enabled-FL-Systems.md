HBFL-CCFD is a framework that enables multiple banks to collaboratively train fraud detection models without sharing sensitive transaction data. It integrates a two-tier hierarchical federated learning architecture with blockchain for auditability, Differential Privacy for secure local training, and IPFS for efficient off-chain model storage.
